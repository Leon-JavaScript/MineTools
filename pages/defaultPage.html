Error behavior

Status Error When it happens
400 BAD_REQUEST Missing identifier on /uuid or invalid UUID format on /profile/{uuid}.
404 NOT_FOUND Unknown route, or username/UUID does not exist upstream.
429 TOO_MANY_REQUESTS Rate limiter blocks a valid API route request.
500 INTERNAL_ERROR Unhandled worker error or upstream failure.
